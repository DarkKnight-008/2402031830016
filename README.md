A simple Python mini project that visualizes weekly weather data using Matplotlib Library.

This project displays:
- Line chart of temperature over a week
- Bar chart of humidity
- Pie chart showing weather condition distribution (Sunny, Rainy, Cloudy)

---

# Features
- Visualizes weather data in 3 types of plots
- Clean and simple charts with labels, titles, and colors
